NBA App

Project information

  • Technologies / Frameworks used: HTML, CSS, Bootstrap, and Vue.js
  • Github URL: https://github.com/dlopez0882/nbaapp
  • Project URL: https://dlopez0882.github.io/nbaapp

A Single Page Application (SPA) that renders data about NBA players, games, and box-scores - all for past and present seasons. Utilizes Vue.js for to render component-based experiences, as well as Vue Router to enforce SPA architecture.

App data is provided via a 3rd party API (balldontlie).
